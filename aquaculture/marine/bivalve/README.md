This calculates the pressures for bivalves.

First run **STEP1_fao_bivalve_production.Rmd** to prep the FAO data for bivalve production.

The order of the folders does not matter. The production per farms raster layer that is used is from the aquaculture mapping project: https://github.com/OHI-Science/aquaculture_mapping (Clawson et al. 2022).
